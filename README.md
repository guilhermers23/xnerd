# Projeto Final Curso EBAC

## Requisitos do Projeto

Seu projeto deverá atender aos seguintes critérios:

#### Sistema de Autenticação e Criação de Conta

- O sistema deve permitir que novos usuários se cadastrem e façam login com segurança.

#### Configuração de Perfil

- O usuário pode alterar sua foto de perfil, nome e senha. Nenhuma alteração deve ser obrigatória para o usuário, ele pode ter a liberdade de escolher o que deseja alterar.

#### Sistema de Seguir e Feed de Notícias

- Possibilidade de seguir outros usuários e ver sua lista de seguidos e seguidores.

- O feed de notícias deve exibir postagens apenas das pessoas seguidas.

#### Interações nas Postagens

- As postagens podem receber curtidas e comentários de outros usuários.

#### Deploy e Entrega Final

- A aplicação deve estar hospedada e acessível online.

- O código-fonte deve ser entregue via GitHub, com instruções claras de uso, além do link do deploy.

### Tecnologias obrigatórias

- Back-end: Python com Django, seguindo arquitetura REST.

- Banco de dados: livre escolha (SQLite, PostgreSQL, etc).

- Front-end: React, Vue, HTML/CSS puro, ou, caso faça um projeto monolítico, os templates devem ser integrados ao projeto.
